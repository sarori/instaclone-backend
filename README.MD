## Features

### User

-   [x] Create Account
-   [x] See Profile
-   [x] Login
-   [x] Edit Profile
-   [x] Change Avatar (Image Upload)
-   [x] Follow User
-   [x] Unfollow User
-   [x] See Followers and See Following w/ Pagination
-   [x] Computed Fields
-   [x] Search Users

### Photos

-   [x] Upload Photo
-   [x] See Photo
-   [x] See Hashtags
-   [x] Search Photos(by caption)
-   [x] Edit Photo
-   [x] Like / Unlike Photo
-   [x] See Photo Likes
-   [x] See Photo Comments
-   [x] See Feed
-   [x] Delete Photo

### Comments

-   [x] Comment on Photo
-   [x] Delete Comment
-   [x] Edit Comment

### Extras

-   [x] S3 Image Upload

### DMs

-   [x] See Rooms
-   [x] Send Message (Create Room)
-   [x] See Room
-   [x] Computed Fields
-   [x] See message
-   [x] Realtime Messages
